AI 配置说明

在 ~/.osint/config.yaml 的 ai 段可配置：

  • persona_inject: true — 搜罗/追问/简报注入画像
  • dwell_save_no_ai: true — 高停留自动收录跳过 AI（省 API）
  • auto_persona_rebuild: prompt — off / prompt（横幅提示）/ auto（静默重建）
  • auto_persona_rebuild: auto — 达阈值后自动重建，仅提示「已自动重建」（prompt=手动提醒，off=关闭）
  • auto_persona_rebuild_threshold: 50 — 自上次构建后新增行为条数达阈值触发

在 search 段可配置：

  • include_slurs: true — 搜罗时默认扩展贬义/黑称关联词
  • max_expanded_queries: 8 — 单次搜索最多并行查询数
  • comment_mine_top: 12 — 默认对 top N 条结果拉取热评/字幕/弹幕
  • discover_sources: [bilibili, zhihu, web, v2ex] — 联网发现关联词的探针来源
  • persist_discovered_aliases: true — 自动写入 ~/.osint/entities/discovered.yaml

实体词表：将 YAML 放入 ~/.osint/entities/，参见仓库 docs/examples/entities/；联网发现词会自动沉淀到 discovered.yaml
