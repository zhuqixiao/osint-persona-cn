# OSINT 浏览器扩展（MV3）

**安装后日常上网即可**，数据仅写入本机 `http://127.0.0.1:8787`。

## 工作方式

| 模式 | 说明 |
|------|------|
| **被动采集**（默认开） | 访问支持站点时自动记录 URL、标题、停留时长 |
| **API 拦截** | B 站/知乎/GitHub 页面内部分页 API 自动截获（点赞、收藏、赞同等） |
| **定时后台同步**（默认开） | 每 4 小时调用本机服务端 API 拉取 B站/知乎账号数据（Cookie） |
| **右键收录** | 写入行为事件 + 调用 `/api/save` 进知识库 |
| **高停留自动收录** | 内容页停留 ≥90 秒自动进知识库（`~/.osint/config.yaml` 可调 `extension.dwell_save_ms`） |

## 支持平台

B站、知乎、GitHub、V2EX、掘金、少数派、虎嗅、36氪、小红书、微博、豆瓣、Twitter/X

## 安装

1. `osint web` 启动 Web
2. Edge/Chrome → 扩展 → 开发者模式 → **加载已解压** → 选 `extension/`
3. 弹窗保持「启用采集」「被动浏览」「定时后台同步」勾选

之后正常刷社交即可；扩展每分钟把队列 flush 到本机。

## 手动操作（可选）

- **立即同步队列**：把内存中待发送事件写入本机
- **服务端拉取 B站/知乎**：立刻调用本机 `/api/ingest/bilibili` 与 `/api/ingest/zhihu`（知乎浏览记录走 Cookie API；`/footprints` 页面已下线）
- Web「扩展与导入」页也有 **一键服务端拉取**，不打开扩展弹窗也能用

## 与 API 导入的关系

- **API 导入**：一键 bulk（观看/收藏 500 条）
- **扩展**：日常增量 + API 补不全的部分（赞同明细、更多点赞、跨站浏览）

推荐：安装扩展后偶尔在「行为导入」页点一次 API 导入做全量校准。

## 隐私

- 只采集本浏览器、本账号访问的页面
- 不上传云端；弹窗可随时关闭采集
